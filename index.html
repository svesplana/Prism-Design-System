Why Prism

Everything in one system

Prism gives designers and developers a shared language — from tokens to production code.

Design Tokens

Color, spacing, typography, and motion defined as single-source-of-truth tokens shared between Figma and code.

25+ Components

Buttons, inputs, modals, carousels, accordions — every component you need with full variant coverage.

Figma Plugin

Install our Figma plugin to scaffold a complete design system in minutes — styles, components, and icons ready to use.

Multi-Theme

Three distinct product themes — SparkRamp orange, StreamOps blue, ClarityMap violet — sharing the same component API.

Lucide Icons

Over 30 crisp SVG icons from Lucide (MIT license) baked into the system — with a full icon library page in Figma.

Open Source

MIT licensed. Use it freely for personal and commercial projects. Contributions welcome on GitHub.
